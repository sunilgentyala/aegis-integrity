Ten Modules. Zero Blind Spots.

Each module runs independently and contributes to a unified risk score. Disable any module via PipelineConfig.

Module 01
Citation Hallucination Detection
Resolves every DOI via the Crossref REST API. Cross-checks author, year, and title against the live record. Studies show over 50% of AI-generated reference lists are fabricated. No other open-source tool does this.

Verdicts: VALID | MISMATCH | HALLUCINATED | UNRESOLVABLE | NO_DOI
Module 02
LLM Watermark Detection NEW v2.0 IT
Kirchenbauer (2023) green-list z-test without the watermark key. Detects watermarked GPT-4, Gemini, and other LLM output. False positive rate below 1 in 1,000,000 at z>4.0. Also checks token entropy and rank skew for soft watermarks.
Module 03
Citation Network Analysis NEW v2.0
Analyzes the full reference list: self-citation inflation (>30% threshold), predatory journal pattern matching, year-clustering (LLM hallucination signature), OpenAlex quality lookup, and missing DOI rate analysis.
Module 04
ESL-Calibrated AI Content Detection
Fixes the 61.3% false-positive rate on non-native English writers (Stanford 2023). Per-language threshold multipliers for 15 languages. Paragraph-level scoring. GPT-2 perplexity + burstiness + cross-perplexity ratio + stylometric ensemble.
Module 05
Semantic Coherence Analysis NEW v2.0 IT
Catches AI-polished text that passes perplexity filters. Measures discourse connector density, sentence length CV, MTLD diversity, epistemic hedging rate, and section template matching. Targets the "too smooth to be human" signature.
Module 06
Semantic / Paraphrase Plagiarism
SBERT dense retrieval + CrossEncoder reranking catches concept-level paraphrase where no exact words are shared. BM25 / TF-IDF tools miss this. Model: paraphrase-MiniLM-L6-v2 (80 MB). Index: FAISS IndexFlatIP.
Module 07
N-Gram Plagiarism (MinHash LSH)
Dual index: word 3-gram (verbatim copy) and character 5-gram (obfuscation via typos or Unicode substitution). 128 MinHash permutations. Sub-linear query time over large corpora via LSH banding.
Module 08
Stylometric Ghostwriting Detection
Burrows' Delta with 60-dimensional feature vectors (10 scalar + 50 function-word frequency dimensions). Segments with Delta >0.40 from the document baseline are flagged as potential ghostwritten sections.
Module 09
Self-Plagiarism / Text Recycling
Three-layer detection: character 5-gram Jaccard (verbatim), word 3-gram Jaccard (near-verbatim), SBERT cosine ≥0.88 (cross-language recycling). Risk levels follow COPE text recycling guidelines at 15% / 30% thresholds.
Module 10
Batch / Classroom Analysis NEW v2.0 IT
Cross-document essay mill detection. Pairwise MinHash similarity matrix + rare vocabulary overlap + section sequence fingerprinting. Union-Find clustering groups submissions by suspected shared AI source. Scales to hundreds of documents.

Install in 3 Steps

All 10 detectors active after a single pip install. Minimal install (no PyTorch, no SBERT) also available.

1

Clone the repo

git clone https://github.com/sunilgentyala/aegis-integrity.git

cd aegis-integrity
2

Install with all modules

pip install -e "[ml,nlp,bib]"

python -m spacy download en_core_web_sm
3

Run your first analysis

aegis analyze paper.pdf --html report.html
▲

Or use Docker (recommended for production)

docker compose up --build 
# Single paper analysis
aegis analyze paper.pdf --html report.html

# Against a reference corpus
aegis analyze paper.pdf \
  --corpus ./corpus/ --html report.html

# Classroom batch scan (NEW v2.0)
aegis batch ./submissions/ \
  --html batch_report.html

# Pairwise comparison
aegis compare draft_v1.pdf draft_v2.pdf

# Start REST API (Swagger at /docs)
aegis serve --host 0.0.0.0 --port 8000

from aegis.core.pipeline import AEGISPipeline, PipelineConfig

cfg = PipelineConfig(
    citation_email="you@org.com",
    run_watermark_detector=True,
    run_citation_network=True,
    run_coherence_analyzer=True,
)
pipeline = AEGISPipeline(config=cfg)
report = pipeline.analyze("submission.pdf")
print(report.overall_risk)  # LOW|MEDIUM|HIGH|CRITICAL
print(report.watermark_result)
print(report.coherence_result)
