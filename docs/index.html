Ten Modules. Zero Blind Spots.

Every module runs independently and contributes to a unified risk score. Skip any you don't need via PipelineConfig.

Module 01
Citation Hallucination Detection
Resolves every DOI via Crossref REST API. Cross-checks author, year, and title against the live publication record. Catches the >50% fabrication rate in AI-generated reference lists. No other open-source tool does this.
Module 02
LLM Watermark Detection NEW v2.0
Implements Kirchenbauer (2023) green-list / red-list z-test without the watermark key. Detects watermarked GPT-4, Gemini, and other LLM output. False positive rate < 1 in 1,000,000 at z > 4.0. Also checks entropy and token rank skew for soft watermarks.
Module 03
Citation Network Analysis NEW v2.0
Analyzes the full reference list structure: self-citation inflation (>30% threshold), predatory journal pattern matching, year-clustering (LLM fabrication signature), OpenAlex quality lookup, and missing DOI rate analysis.
Module 04
ESL-Calibrated AI Content Detection
Fixes the 61.3% false-positive rate on non-native English writers (Stanford 2023 study). Per-language threshold multipliers for 15 languages. Paragraph-level scoring pinpoints injected AI sections. GPT-2 perplexity + burstiness + cross-perplexity ratio + stylometric ensemble.
Module 05
Semantic Coherence Analysis NEW v2.0
Catches AI-polished text that passes perplexity filters. Measures discourse connector density ("Furthermore", "Moreover"), sentence length uniformity (CV < 0.35 = suspicious), epistemic hedging rate, MTLD diversity score, and section template matching.
Module 06
Semantic / Paraphrase Plagiarism
SBERT dense retrieval + CrossEncoder reranking catches concept-level paraphrase where no exact words are shared. BM25/TF-IDF-only tools miss this entirely. Model: paraphrase-MiniLM-L6-v2 (80 MB). Index: FAISS IndexFlatIP.
Module 07
N-Gram Plagiarism (MinHash LSH)
Dual index: word 3-gram (verbatim copy) and character 5-gram (obfuscation via typos or Unicode substitution). 128 MinHash permutations. Sub-linear query time over large corpora via LSH banding.
Module 08
Stylometric Authorship Profiling
Burrows' Delta with 60-dimensional feature vectors (10 scalar + 50 function-word frequency dimensions). Segments with Delta > 0.40 from document baseline are flagged as potential ghostwritten sections. Catches essay mills that mix human and AI writing.
Module 09
Self-Plagiarism / Text Recycling
Three-layer detection: character 5-gram Jaccard (verbatim), word 3-gram Jaccard (near-verbatim), SBERT cosine ≥ 0.88 (cross-language paraphrase recycling). Risk levels follow COPE text recycling guidelines at 15% / 30% thresholds.
Module 10
Batch / Classroom Analysis NEW v2.0
Cross-document essay mill detection. Pairwise MinHash similarity matrix + rare vocabulary overlap + section sequence fingerprinting. Union-Find clustering groups submissions by suspected shared AI source. Statistically catches when a class all used the same AI prompt.

Get Started in 3 Steps

Full installation with all 10 detectors active. Minimal install (no PyTorch) available for citation + coherence only.

1

Clone and install

git clone https://github.com/sunilgentyala/aegis-integrity.git

pip install -e ".[ml,nlp,bib]"
2

Download spaCy model

python -m spacy download en_core_web_sm
3

Run your first analysis

aegis analyze paper.pdf --html report.html
▲

Or use Docker (recommended)

docker compose up --build 
# Analyze a single paper
aegis analyze paper.pdf \
  --html report.html \
  --output report.json

# Against a reference corpus
aegis analyze paper.pdf \
  --corpus ./corpus/ \
  --html report.html

# Classroom batch scan (NEW v2.0)
aegis batch ./submissions/ \
  --html batch_report.html

# Pairwise comparison
aegis compare draft_v1.pdf draft_v2.pdf

# Start REST API
aegis serve --port 8000
Python API Example
from aegis.core.pipeline import AEGISPipeline, PipelineConfig

cfg = PipelineConfig(
    citation_email="you@university.edu",
    run_watermark_detector=True,   # NEW v2.0
    run_citation_network=True,     # NEW v2.0
    run_coherence_analyzer=True,   # NEW v2.0
)
pipeline = AEGISPipeline(config=cfg)
pipeline.load_corpus([("Smith2023", open("smith2023.txt").read())])

report = pipeline.analyze("submission.pdf")
print(report.overall_risk)            # LOW | MEDIUM | HIGH | CRITICAL
print(report.watermark_result)         # WatermarkResult (v2.0)
print(report.citation_network_result)  # CitationNetworkResult (v2.0)
print(report.coherence_result)         # CoherenceResult (v2.0)
